## Funcionalidad
* Cargar imagen
* Recortar imagen
* Descargar imagen recortada

## Setup
```bash
# Clonar el repositorio
$ git clone https://github.com/frank-hz/image-cropper-vite-js.git

# Ir a la carpeta
$ cd image-cropper-vite-js

# Instalar dependencias
$ npm install

# Ejecutar aplicacion
$ npm run dev
```
